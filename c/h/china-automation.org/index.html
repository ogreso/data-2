  • Description

    中国工业自动化网是中国工业控制及自动化领域的领先权威媒体，是工业互联与智能制造的产业互联网服务平台，面向装备制造业的工业自动化、生产自动化、过程自动化、制造业转型升级,两化深度融合的企业信息化,工业智能化,智能制造,数字智造,工业4.0,中国制造2025,互联网+,提供全面、及时、系统、准确、专业及个性化的信息服务和资料检索。涵括自动化新闻、自动化企业库、自动化产品、产品选型、自动化供求信息、专业展会、行业应用、工控搜索、工控资料下载、专业杂志等服务，同时为自动化企业提供一站式数字微服务平台，充分利用移动互联网的优势，为企业提供各类便捷服务。并以业内的行业精英人士为依托，构建行业高端、绿色生态圈，致力成为业内sns生态圈的缔造者和领导者-中国工业自动化网。
